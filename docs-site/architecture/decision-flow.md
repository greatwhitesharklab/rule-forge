# 决策流引擎

RuleForge 集成 Flowable 8 BPM 引擎，提供基于 BPMN 2.0 标准的决策流编排能力。

## BPMN 2.0 支持

### 支持的节点类型

| BPMN 元素 | 对应功能 | 说明 |
|-----------|---------|------|
| Start Event | 开始节点 | 流程入口 |
| End Event | 结束节点 | 流程出口 |
| Service Task | 规则任务 | 执行规则集/决策表/评分卡 |
| Exclusive Gateway | 排他网关 | 条件分支 |
| Parallel Gateway | 并行网关 | 并行执行 |
| Script Task | 脚本任务 | 执行自定义脚本 |

### 支持的连线类型

- **顺序流 (Sequence Flow)** — 默认执行顺序
- **条件流 (Conditional Flow)** — 带条件表达式的分支

## 执行流程

1. 解析 BPMN XML 定义
2. 部署到 Flowable 引擎
3. 启动流程实例
4. 依次执行节点
5. 在规则节点中调用 RETE 引擎
6. 根据条件流选择分支
7. 到达结束节点，收集结果

## 与规则引擎的集成

决策流中的每个「规则任务」节点对应一个规则文件：
- 规则集 → RETE 会话执行
- 决策表 → 条件匹配
- 评分卡 → 加权计算

多个规则节点通过流程编排串联，实现复杂的业务决策逻辑。
