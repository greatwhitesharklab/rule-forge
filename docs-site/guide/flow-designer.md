# 决策流设计器

RuleForge 内置基于 BPMN 2.0 标准的决策流设计器，支持可视化编排复杂业务流程。

## 核心概念

### 节点类型

| 节点 | 说明 | 图标 |
|------|------|------|
| 开始节点 | 流程入口 | ⬤ |
| 结束节点 | 流程出口 | ◯ |
| 规则任务 | 执行规则集/决策表/评分卡 | ▭ |
| 排他网关 | 条件分支（只走一条） | ◇ |
| 并行网关 | 并行执行（多条同时走） | + |
| 服务任务 | 调用外部服务/数据源 | ▭ |

### 连线

节点之间的连线可以设置条件表达式，用于控制流程走向。

## 创建决策流

1. 在项目中点击「新建规则」→ 选择「决策流」
2. 从左侧工具栏拖拽节点到画布
3. 连接节点，设置条件表达式
4. 为每个规则节点配置引用的规则文件
5. 保存并测试

## 最佳实践

- 每个决策流应有一个明确的主路径
- 使用排他网关处理异常分支
- 复杂子流程可以拆分为独立的规则集
- 利用变量库统一管理输入输出
